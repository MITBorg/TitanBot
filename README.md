# TitanBot #

An IRC bot developed for the moparisthebest community using Java and Maven.

## Modules
* Weather
* Urban Dictionary (Lookup)
* (Last) Seen
* Memo
* Sed (Replacement)
* Stats
* TestCommand
* FlameBot
* Help
